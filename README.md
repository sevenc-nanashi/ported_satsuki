# ported_satsuki

[さつき様のスクリプト](https://dic.nicovideo.jp/u/2787743)をAviUtl2用に移植するプロジェクトです。

> [!WARNING]
> このプロジェクトのスクリプトはある程度速く動作することを目指していますが、専用のスクリプトやプラグインに移行することを推奨します。
> （自分一人で[この量](./TASKLIST.md)のスクリプトをすべて業界（？）最速レベルまで最適化するのは現実的ではないため）

## 差異

- 切れ込み：単色画像以外にも対応する方式に変更
- しきい値フィルタ：「これ以降のエフェクトを適用する・しない」という方式に変更
- 以下のエフェクトはカスタムオブジェクトになりました：
  - MAGI_A_object、MAGI_B_object、MAGI_C_object、magicA\_大円
  - ADC\_扇状、ADC\_図形
- TA位置(pos)関数、TA位置(pos2)関数：スクリプト制御ではなくオブジェクト内の`text`パラメーターで指定する方式に変更
- TA円形：「外向き」で外向きになるように
- 以下のエフェクトは統合されました：
  - ランダム配置表裏(A・B)
  - パーティクル表裏(A・B)
  - 画像ループMOV_A・B
  - 連番画像ループ[A・B]
  - 連番円形並べ[A・B]
  - 連番円形-カメラ[A・B]
  - 連番ランダム配置[A・B]
  - 連番パーティクル[A・B]
  - 連番球状配置[A・B]
  - 連番らせん[A・B]
  - TAテキスト配置(A・B)
